    1. 테이블 <table></table>
      ① 테이블의 기본 구조(table, tr, td)
        tr : 세로쪽 셀, td : 가로쪽 셀

      ② 테이블만들기
        <table border="1">
        <tr>
        <td>TD: Table Data (1열 1행)</td>
        <td>테이블의 가로줄, 즉 행과 관계가 있다. (1열 2행)</td>
        </tr>
        <tr>
        <td>TR: Table Row (2열 1행)</td>
        <td>테이블의 세로줄, 즉 열과 관계가 있다. (2열 2행)</td>
        </tr>
        </table>
        HTML 적용 시
        

          

          TD: Table Data (1열 1행)
          테이블의 가로줄, 즉 행과 관계가 있다. (1열 2행)
          

          

          TR: Table Row (2열 1행)
          테이블의 세로줄, 즉 열과 관계가 있다. (2열 2행)

    3. 테이블 <td></td>
      ① td 태그의 속성들 : align=”“로 가로 정렬, valign=”“로 세로 정렬,
        background=“”로 그림 넣기, bgcolor=“”로 배경색 지정,
        width=“”와 height=“”로 크기 지정
        <table border="1" bordercolor="skyblue" width="400" height="200">
        <tr>
        <td valign="top" bgcolor="#46CCFF" height="50">왼쪽 상단 정렬</td>
        </tr>
        <tr>
        <td align="center" background="sample.jpg">가운데 정렬(기본값)</td>
        </tr>
        <tr>
        <td align="right" valign="bottom">오른쪽 하단 정렬</td>
        </tr>
        </table>
        HTML 적용 시
        

          

          왼쪽 상단 정렬
          

          

          가운데 정렬(기본값)
          

          

          오른쪽 하단 정렬
          

          

        align 값 : left, center, right
          기능 : 셀 안에 있는 데이터의 정렬방식 지정, 기본값=left
        valign 값 : top, middle, bottom
          기능 : 셀 안에 있는 데이터의 세로위치 지정, 기본값=middle
        background 값 : 이미지 URL
          기능 : 각 셀의 배경화면 지정
        bgcolor 값 : RGB 코드 또는 색이름
          기능 : 각 셀의 배경색 지정
        width 값 : 픽셀값 또는 %
          기능 : 셀의 넓이 지정
          예 : td에서의 width 속성은 한 행 전체의 넓이를 일정한 크기로 설정
        height 값 : 픽셀값 또는 %
          기능 : 셀의 높이 지정
