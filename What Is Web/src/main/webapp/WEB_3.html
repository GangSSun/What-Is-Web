Web이란? Web의 동작구조 Web Programming의 구조 백 엔드 중심 개발이란? 프론트 엔드 중심 개발이란? 

      웹 프로그래밍을 이루는 구성 요소에는 클라인언트, 서버, 프론트엔드, 백엔드 등 수없이 많다.
      
      먼저 클라이언트와 서버에 관해 알아보도록 하자.

      웹 서비스를 제공하기 위해선 서버 컴퓨터와 서버에서 동작하는 여러 프로그램이 필요한데,
      이때 서버에서 동작하는 프로그램 영역을 백엔드라고 한다.
      
      서비스를 이용하는 쪽을 클라이언트라고 하는데, 클라이언트에서 웹 서비스를 이용하려면
      웹 브라우저라고 하는 S/W가 필요하며 클라이언트에서 동작하는 프로그램 영역을 프론트 엔드라고 한다.
