CSS란? CSS 사용 예시 

      CSS는 HTML과 함께 웹을 구성하는 기본 프로그래밍 요소인데,
      HTML이 텍스트나 이미지, 표와 같은 구성요소를 웹 문서에 넣어 뼈대를 만드는 것이라면,
      CSS는 글씨의 색상이나 크기, 이미지 크기나 배치 방법 등 웹 문서의 디자인 요소를 담당한다.
      CSS의 특징은
      1. CSS 구문은 선택자와 선언부로 구성된다.
      2. 선택자는 디자인을 적용하고자 하는 HTML 요소이므로 선택자 정의가 중요하다.
      3. 선언부는 {} 블록을 사용하며 다수의 속성을 포함한다.
      4. 각 속성 정의는 '속성:값;' 형식이며 항상 세미콜론(;)으로 끝내야 한다.

      CSS 사용 장점은 다음과 같다.
      1. 웹 문서의 내용과 별개로 디자인만 바꾸거나, 디자인은 그대로 두고 웹문서의 내용 변경이 용이하다.
      2. 다양한 기기(PC, 스마트폰 등)에 맞게 탄력적으로 디자인이 바뀌도록 반응형 디자인을 구현할 수 있다.
      3. 동일한 문서 구조여도 서로 다른 CSS 테마 적용이 가능하다.

      CSS를 HTML에 적용하기 위한 방법에는 다음의 세가지가 있다.
      1. 인라인 스타일 시트 : 말 그대로 HTML 태그에 CSS 속성을 정의하는 것
      2. 내장 스타일 시트 : HTML 문서의 <head> 부분에 CSS 정의 부분을 포함하는 것.
      3. 외장 스타일 시트 : 별도의 CSS 파일을 생성한 후 HTML 문서에 링크로 포함한다. 즉 하나의 파일로 여러 문서에 적용이 가능하다.
