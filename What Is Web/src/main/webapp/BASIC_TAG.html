      ② 문단 정렬 속성 <h# align=“”>
        <h1>제목 표시(h1)</h1>
        <h2 align="center">제목 표시(h2)</h2>
        <h3 align="right">제목 표시(h3)</h3>
        <h4 align="left">제목 표시(h4)</h4>
        <p align="center">두 번째</p>
        HTML 적용 시
        
        
제목 표시(h1)

        
제목 표시(h2)

        
제목 표시(h3)

        
제목 표시(h4)

        
두 번째

        

        문단 정렬 기본은 왼쪽 정렬
        left : 왼쪽 정렬(기본값)
        center : 가운데 정렬
        right : 오른쪽 정렬
        align 속성 사용시 <p>와 </p>를 쌍으로 사용

      ③ 줄바꾸기 : <br>

      ④ 문단구분 : <p>
        <p> </p> 태그
        <br> 태그는 반복 효과 가능하지만 <p> 태그는 반복 효과 없다. <br> 태그 두 번이 <p> 태그 한번 효과를 나타낸다. 

      ⑤ 문단 가운데 정렬 <center></center>

      ⑥ 수평선 삽입
        <hr align="left" width="90%" size="5" color="black" noshade>
        HTML 적용 시
        

        <hr> 태그의 속성
        align(left,center,rigth) : 수평선의 위치 지정
        width : 수평선의 넓이 지정(% 또는 픽셀로 설정)
        size : 수평선의 높이를 픽셀로 설정
        color : 수평선의 색상 지정
        noshade : 그림자 없는 평면적인 수평선 지정
        예 <hr align="center" width="50%" size="2" color="red" noshade>

      3. 문서를 일정한 모양으로 보여주는 태그
      ① <pre> </pre> 태그
        <pre>
        웹프로그래밍은 어렵지 않아요. 잘 따라오세요.</pre>
        HTML 적용 시
        
          웹프로그래밍은 어렵지 않아요. 잘 따라오세요.

        <pre> 태그는 미리 설정된 모양대로 브라우저에 출력
        <pre> 태그 내에서도 다른 태그들은 본래의 기능을 수행
        <pre> 태그 텍스트는 동일한 너비의 글꼴로 출력

      ② <pre> 태그를 이용한 코드 출력
        예약 문자를 사용한 HTML
        <pre>
        <html>
        <head>
        <title>HTML 문서의 기본 형태</title>
        </head>
        <body>
        이곳에 본문 내용을 입력합니다. </body>
        </html>
        </pre>

        예약 문자를 사용하지않고 HTML 적용 시
        
          
          
          HTML 문서의 기본 형태
          
          
          이곳에 본문 내용을 입력합니다. 
          
          

        태그 삽입시 예약문자를 사용해야 태그를 온전히 표현할 수 있음
