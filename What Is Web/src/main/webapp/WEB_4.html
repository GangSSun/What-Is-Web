Web이란? Web의 동작구조 Web Programming의 구조 백 엔드 중심 개발이란? 프론트 엔드 중심 개발이란? 

      백엔드 중심 개발은 전통적인 웹 개발 모델로 서버에서 모든 것을 담당하는 방식이다.

      하지만 최근엔 프론트 엔드 중심의 개발이 인기를 끌고 있는데 백엔드 중심 개발의 문제점들 때문이다.

      가장 큰 문제점은 모바일 네트워크의 속도가 느리며 이용에 많은 비용을 지불해야 한다는 것 때문 인데
      즉 서버로 부터 가져와야 하는 데이터가 많아졌기 때문에 과거에 비해 더많은 비용과 많은 시간을 소비해야 한다는 뜻이다.

      또 서버에 화면 갱신을 요청하게 될 경우 모든 데이터가 다시 전송 되어야 해서 모바일 환경에는 부적합하다는 점이있다.
