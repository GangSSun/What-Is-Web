Web이란? Web의 동작구조 Web Programming의 구조 백 엔드 중심 개발이란? 프론트 엔드 중심 개발이란? 

      월드 와이드 웹(World Wide Web)을 우리는 줄여서 WWW 혹은 웹이라고 하는데, 
      이 웹은 인터넷에서 운영되는 서비스 중 하나로, 웹 자체가 인터넷을 의미하는 것은 아니다.

      웹을 처음 개발한 사람은 1989년 스위스에 있는 유럽 입자 물리 연구소의 팀 버너스 리로,
      최초의 개발 목적은 연구자들이 흩어져 있는 정보를 손쉽게 정리하고 공유하기 위해 개발 되었다.

      즉 HTML(HyperText Markup Language)이라는 간단한 마크업 언어를 통해 정보의 연결이 가능하도록
      하이퍼 텍스트를 포함한 콘텐츠를 제공하며, HTTP라는 프로토콜을 사용해 
      TCP/IP 네트워크 상에서 서비스를 운영하게 된 것을 웹의 시작이라고 볼 수 있다.

      웹은 다음과 같은 특징을 가지고 있다.
      1. 인터넷은 컴퓨터 네트워크망을 의미하고, 웹은 인터넷 서비스 중 하나를 의미한다.
      2. 웹은 인터넷상의 정보를 하이퍼텍스트 방식과 멀티미디어 환경에서 검색할 수 있게 해주는 정보 검색 시스템이다.
      3. 웹은 HTTP 프로토콜을 사용하여 HTML로 작성된 문서를 연결하여 다양한 콘텐츠를 제공한다.
      4. 웹 서비스는 많은 사람들이 정보를 쉽게 공유하고 접근할 수 있도록 하는 것을 목적으로 한다.
