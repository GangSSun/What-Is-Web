meta

    <meta> 태그는 해당 문서에 대한 정보인 메타데이터(metadata)를 정의할 때 사용한다.

    <meta> 요소는 <base>, <link>, <script>, <style>, <title> 요소와 같은 
    다른 메타데이터 관련 요소들이 나타낼 수 없는 다양한 종류의 메타데이터를 제공할 때 사용되며, 
    이렇게 제공된 정보는 브라우저나 검색 엔진, 다른 웹 서비스에서 사용하게 된다.
    이러한 <meta> 요소는 언제나 <head> 요소 내부에 위치해야 한다.
    만약 name 속성이나 http-equiv 속성이 명시되었다면 반드시 content 속성도 함께 명시되어야 하며, 
    반대로 두 속성이 명시되지 않았다면 content 속성 또한 명시될 수 없다.
    
    [예시]
    검색엔진
    <meta name="Keywords" content="web, html, jsp" />
    문자코드 설정
    <meta http-equiv="Content-Type" content="text/html; charset=utf-8" />
    바뀐 주소로 자동 이동
    <meta http-equiv="refresh" content="5;url=http://naver.com">
    <meta> 태그는 닫을 필요 없다. <head>와 </head> 사이에 위치
    http-equiv="항목명" 웹 브라우저가 서버에 명령을 내리는 속성
    content="정보값" meta 정보의 내용을 지정한다. name="정보 이름" meta 정보의 내용을 지정한다. 
    charset="인코딩 언어" 페이지의 문자 인코딩을 선언한다. 값으로는 utf-8 이 들어간다.
